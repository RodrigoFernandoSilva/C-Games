# Running
## Descrição
Running como o própio nome já diz é um runner onde o objetivo é desviar dos obstáculos e conseguir a maior pontuação possivel.
  
## Funcionalidades
* Salva a maior pontuação;
* Os obstaculos vem por cima e por baixo.

## Gênero
* Casual;
* 2D.
