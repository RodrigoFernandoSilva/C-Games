# ShootongGame
## Descrição
ShootongGame é uma copia do jogo space invaders.
  
## Funcionalidades
* Salva a maior pontuação;
* Dificuldade progressiva.

## Gênero
* Casual;
* 2D.

## Linguagem
C

## Biblioteca
SDL
