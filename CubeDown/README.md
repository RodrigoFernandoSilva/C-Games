# CubeDown
## Descrição
CubeDown é um jogo inspirado no tetris, porém com o diferencial de que ao invez de juntar formas de junta cores.
  
## Funcionalidades
* Juntou 4 cores iguais elas se quebram;
* Salva a maior pontuação;

## Gênero
* Casual;
* Puzzle.

## Linguagem
C

## Biblioteca
SDL
