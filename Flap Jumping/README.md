# Flap Jumping
## Descrição
Flap Jumping é um jogo ispirado no FlappyBird, entretando não se tem uma fisica muito parecida.
  
## Funcionalidades
* Salva a maior pontuação;

## Gênero
* Casual;
* 2D.
